# 🎓 Academy Manager

학원 운영에 필요한 **학생 / 수업 / 출결 / 수납 / 강사 관리**를 통합적으로 제공하는  
**웹 기반 학원 관리 시스템**입니다.

> 과제용 SQLite 프로젝트의 한계를 개선하고,  
> **PostgreSQL + Docker 기반 실무 환경**으로 재설계한 프로젝트입니다.

---

## 📌 프로젝트 개요

- **프로젝트명**: Academy Manager
- **개발 목적**:  
  계획서 기반 설계를 통해 유지보수성과 확장성을 갖춘 학원 관리 시스템 구축
- **개발 형태**: 개인 프로젝트
- **개발 환경**: Antigravity Editor

---

## 🛠 기술 스택

### Backend
- Python 3.x
- Django 4.x
- PostgreSQL

### Frontend
- Django Template
- Bootstrap 5 (Dark Mode)

### DevOps / Tools
- Docker
- Docker Compose
- Git / GitHub

---

## 🧩 주요 기능

### 👤 사용자 관리
- 관리자 / 강사 계정
- 로그인 / 로그아웃
- 권한 기반 접근 제어

---

### 🎓 학생 관리
- 학생 등록 / 수정 / 삭제
- 학부모 연락처 관리
- 재원 상태 관리 (재원 / 휴원 / 퇴원)
- 반 배정
- 학생 메모

---

### 🏫 반 / 수업 관리
- 반 생성 및 수정
- 수업 요일 / 시간 설정
- 담당 강사 지정
- 반별 학생 목록 조회

---

### 🕒 출결 관리
- 출석 / 결석 / 지각 / 조퇴 기록
- 날짜별 출결 조회
- 출결 메모 관리

---

### 💳 수납 관리
- 수강료 등록
- 월별 수납 내역 관리
- 납부 상태 확인 (미납 / 완납)
- 결제 방식 기록

---

### 👨‍🏫 강사 관리
- 강사 정보 관리
- 담당 반 조회
- 강사별 수업 정보 확인

---

### 📊 관리자 대시보드
- 재원 학생 수
- 월별 매출 현황
- 반별 출석률
- 미납자 목록
